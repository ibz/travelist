Formatting

  1. Separate paragraphs by one or more blank lines.
  2. To create a heading, use dashes to underline it. Like this: 
    My Title
--------

  3. To create a list, use asterisks at the beginning of every element. Like this: 
    * First thing
* Second thing
* Yet another one
  4. To create a link, use the following syntax:
    [Google homepage](http://google.com/)
  5. To create italic or bold text, use: *italic* and **bold** respectively.
  6. Inline HTML is not allowed.
